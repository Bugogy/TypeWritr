								Welcome to Darkscrawl
										by
								    Cody Azevedo

Darkscrawl is an open-source, simple, and menacingly dark text editor. Inspired by
emacs, personalized for the dark and minimal mind.

-VERSION-
0.9 beta

-CONTROLS-
C-O: Open file browser.
C-S: Save file.
C-N: New file. You can also create a new file in the directory browser.
C-F: Find in file. RET returns next result, if it exists.
C-G: Exit top browser bar / command line.
C-X: Split the window.
