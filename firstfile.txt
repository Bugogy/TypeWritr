								Welcome to Darkscrawl
										by
								    Cody Azevedo

Darkscrawl is an open-source, simple, and menacingly dark text editor.
This editor follows the UNIX design philosophy.
Feel free to fork this directory and customize it to your liking!

-VERSION-
0.9 beta

-CONTROLS-
C-O: Open file browser. Press ESC to close the browser.
C-S: Save file.
C-N: New file. You can also create a new file in the directory browser.
C-M: Switch editing mode. [ins] allows for file editing. [nav] sets buffer as read only.
C-F: Find in file. RET returns next result, if it exists.
C-E: Move to end of line
C-S-E: Select to end of line
C-G: Exit top browser bar / command line.
C-X: Split the window.

-THEME-
To change to a basic color theme, use:

	> python darkscrawl.py <color>

Current color options:
red, orange, yellow, blue, green
(Green is my favorite!)
